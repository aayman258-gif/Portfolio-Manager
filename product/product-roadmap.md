# Roadmap

Building on: PyPortfolioOpt (optimization), FinanceToolkit (fundamentals), regime detection concepts, yfinance (data), Streamlit (UI)

## Sections

### 1. Portfolio Position Tracker
Load and track current positions with basic P&L, holdings breakdown, and performance metrics

### 2. Regime Detection + Market Dashboard
Detect current market regime and display macro indicators (reuse regime concepts from options system)

### 3. Unified Scoring Engine
Score each position on quant signals (momentum, volatility, regime fit) and fundamental metrics (growth, quality, margins)

### 4. Optimization & Rebalancing Tool
Generate regime-aware optimal allocation and rebalancing recommendations using PyPortfolioOpt

### 5. Action Dashboard
Single-page view showing regime, portfolio health, actionable recommendations ("What should I do today?")
